Exercise 3 - Convolutions

As we have seen in the videos, we can produce cool effects using the mathematical technique of convolutions. We input one image M and get a new image Mâ€² back.

Conceptually we think of M as a matrix. In practice, in Julia it will be a Matrix of color objects, and we may need to take that into account. Ideally, however, we should write a generic function that will work for any type of data contained in the matrix.

The result of the convolution over a given window, centred at the point (i,j) is a single number; this number is the value that we will use for Mi,jâ€². (Note that neighbouring windows overlap.)
